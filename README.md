# Task List Agent

Status: Completed and verified.

---

## Overview

Task List Agent is an n8n-based automation workflow that stores and manages tasks from chat input and maintains a structured task list.

The workflow supports task updates and spreadsheet automation.

---

## Features

- Chat-based task creation
- Google Sheets integration
- Task storage
- Status updates
- Automation workflow
